## Browserify Boilerplate

A gulp/browserify starter kit, with a few bells and whistles, including
* es6 transpilation
* linting
* live reload
* sass compilation
* image minification
* deployment to gh-pages


### Installation
```bash
npm install -g gulp
npm install
```

### Development

```bash
npm run dev
```

### Deployment

```bash
npm run deploy
```
